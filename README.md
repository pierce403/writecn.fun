# writecn.fun

A tiny Chinese character writing game (React + TypeScript + Tailwind + HanziWriter), built as static HTML/CSS/JS.

## Local dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## GitHub Pages
